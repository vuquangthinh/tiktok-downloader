
Installation
To install the app:

Install Node.js on your computer.

Clone this repository: git clone https://github.com/vuquangthinh/tiktok-downloader.git

Navigate to the project directory: cd tiktok-downloader

Install dependencies: 

```
npm install

```

Start the app: 

```
node index.js

```
